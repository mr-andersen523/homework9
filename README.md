# Node.js Developer Profile Generator (Homework 9)

### How it Works
Using node.js, the application will generate a .PDF developer profile from your GitHub after answering a couple prompt questions.

### Preview
![Demo Screenshot](./demo-pic.png)

### How to Use
1. After opening the index.js file, key "node index.js"
2. The application will prompt for your GitHub username, enter it
3. It will then prompt for a favored color from a short list, select one
4. The application will run and momentarily, you should see a pdf file magically appear!
5. Open that sucker up!
6. VIOLA
